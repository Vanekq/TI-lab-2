# Лабораторная работа №2: Шифрование файлов LFSR

## 🚀 Запуск

Скачайте и запустите `encryption_app.exe` (путь к исполняемому файлу после сборки)

## 📌 Возможности

- **LFSR (Linear Feedback Shift Register)**: 27-битный регистр сдвига с линейной обратной связью
- **Полином**: x²⁷ + x⁸ + x⁷ + x + 1
- Шифрование/дешифрование любых файлов (изображения, документы, текст)
- Автоматическое определение режима по имени файла (`encrypted_*` → расшифровка)

## 📖 Использование

1. Введите двоичный ключ (27 символов, только 0 и 1)
2. Выберите файл для шифрования/дешифрования
3. Нажмите "Зашифровать и сохранить" или "Расшифровать и сохранить"

## 🔐 Особенности

- Режим определяется автоматически:
  - Обычный файл → шифрование → `encrypted_исходное_имя`
  - Файл с префиксом `encrypted_` → расшифровка → `decrypted_исходное_имя`
- LFSR генерирует псевдослучайную последовательность длиной, равной размеру файла
- XOR применяется побайтово к исходным данным и сгенерированному ключу
